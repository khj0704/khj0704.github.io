1) 개발 환경

2) 프로젝트 개요  
 - NFC를 활용한 전자 결제 안드로이드 앱 구현  

3) 프로젝트 구현 기능  
 - NFC 연동 기능  
 - 카드 결제 기능  
 - 결제 내역 관리 기능  
 - 결제 단말기 연동 및 영수증 발급 기능  
 - 서명 저장 및 업로드 기능  
 - 이미지 로드 및 REST API 연동 기능  
 - 커스텀 UI 컴포넌트 구현
