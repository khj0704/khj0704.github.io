1) 개발 환경  
 - Java, eclipse, 삼성 스마트 프린터/복합기  

2) 프로젝트 개요  
 - 삼성 복합기에 내장되는 안드로이드 App 개발  
   (업체 요청사항으로 이미지 및 자세한 내용을 공개할 수 없습니다.)  

3) 프로젝트 구현 기능  
 - 유저 권한(Super/Normal)에 따른 복합기 저장소 접근 기능  
 - 안드로이드 단말기에 있는 파일들과 복합기에 저장소 파일 Print 기능(jpeg,bmp,png,pdf,오피스 파일 등)  
 - WebDAV 연동 기능  
 - 파일 익스플로러 기능  
 - 이미지 편집 기능
