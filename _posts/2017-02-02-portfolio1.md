1) 개발 언어
   - Java xlet, HTML, Javascript

2) 개발 환경
   - eclipse

3) 프로젝트 개요
   - IPTV(ICOD, DnP) 용 광고 애플리케이션
   - 광고주에게 IPTV용 광고 소재 개발 부담을 덜어주기 위한 템플릿형 광고 소재 개발

4) 프로젝트 구현 기능
   - KT QSP 플랫폼 연동 (TimeLine 기반 광고 애플리케이션)
   - 벨소리 등 컨텐츠 다운로드 기능
   - IPTV 결제 기능
